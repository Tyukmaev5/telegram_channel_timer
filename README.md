# Telegram Channel Timer Bot 

Простой Telegram-бот для каналов, который позволяет запускать таймер по указанным **дате, времени и названию**.  
Бот отправляет сообщение с оставшимся временем и автоматически обновляет его до момента завершения таймера.

---

## Что умеет бот

- Создавать таймер на конкретную дату и время
- Показывать оставшееся время в одном сообщении
- Автоматически обновлять сообщение (частота обновления зависит от того, сколько времени осталось)
- Останавливать активный таймер

---

## Команды
- /help
- settimer ГГГГ-ММ-ДД ЧЧ:ММ "Название таймера"
- stoptimer 

## Ограничения
- Максимальная длительность таймера: до 7 дней
- Время берётся из локального времени сервера, где запущен бот

## Стек
- Dart (консольное приложение)
- teledart
